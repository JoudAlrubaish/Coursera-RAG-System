# CourseFinder: Course Recommendation Assistant Using RAG

This repository presents a Retrieval-Augmented Generation (RAG) system that helps users discover and recommend Coursera courses 
using semantic search and Large Language Models (LLMs).

## You will Find Here

- **Data Preprocessing:** Cleaning and preparing the Coursera dataset.
- **Document Chunking:** Converting course records into searchable text chunks.
- **Semantic Embeddings:** Generating vector representations using SentenceTransformers.
- **Vector Database:** Storing and retrieving embeddings with FAISS.
- **LLM Integration:** Generating grounded answers using an OpenAI-compatible model.
- **Gradio Interface:** A simple web interface for asking questions about Coursera courses.

## Skills Demonstrated

- Python
- Pandas & NumPy
- SentenceTransformers
- FAISS Vector Database
- Retrieval-Augmented Generation (RAG)
- Large Language Models (LLMs)
- Prompt Engineering
- Gradio

## Goal

To build an intelligent course recommendation system that combines semantic retrieval with AI-generated responses, demonstrating the core concepts of Retrieval-Augmented Generation (RAG) in a practical application.
